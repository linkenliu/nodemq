# rabbitmq demo

### nodejs amqp 发送消息

### 安装说明:

* npm install

### 文件说明:

* production.js做为生产方(发现消息方)

* consumption.js做为消费方(接收消息方)

### 启动说明

* node ./production

* node ./consumption

### 访问

* http://localhost:8081
